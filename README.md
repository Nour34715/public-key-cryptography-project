# Public-Key Cryptography and Blockchain Security

## Overview

This project demonstrates the implementation and analysis of modern cryptographic systems and blockchain-related security concepts using Python and PARI/GP.

The project explores public-key cryptography, secure voting protocols, cryptographic hash functions, and blockchain-style proof-of-work mechanisms.

## Project Components

### RSA and AES Key Exchange

Generated RSA public/private keys and demonstrated secure encryption and decryption of AES-256 keys.

### ElGamal Secure Voting System

Implemented a simplified secure voting protocol using the homomorphic properties of ElGamal encryption.

### Blockchain Hash Puzzle Simulation

Implemented a SHA-256 proof-of-work style mining puzzle by searching for valid nonces producing hashes with leading zeros.
